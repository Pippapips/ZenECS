# Roadmap

> Docs / Release / Roadmap

Future plans and development roadmap for ZenECS.

## Current Status

**Version:** 1.0.0 (Release Candidate)

**Status:** APIs locked, documentation and samples being finalized.

## Upcoming Releases

### 1.0.0 (Current)

**Focus:** Stable release with complete documentation

**Goals:**
- ✅ Core ECS runtime
- ✅ Unity adapter
- ✅ API documentation
- ✅ Sample projects
- ✅ Complete guides

### 1.1.0 (Planned)

**Focus:** Unity adapter tooling polish

**Planned Features:**
- Enhanced ECS Explorer
- Additional editor tools
- Improved blueprint editor
- Performance profiling tools

### 1.2.0 (Planned)

**Focus:** Snapshot migration helpers

**Planned Features:**
- Migration tooling
- Version compatibility utilities
- Automated migration generation
- Migration testing framework

### 1.3.0 (Planned)

**Focus:** Parallel system scheduler (experimental)

**Planned Features:**
- Multi-threaded system execution
- Parallel query processing
- Thread-safe component access
- Performance optimizations

## Long-Term Vision

### Performance

- **Query optimization**: Faster entity queries
- **Memory optimization**: Reduced allocations
- **Cache optimization**: Better cache utilization
- **Parallel execution**: Multi-threaded systems

### Features

- **Hot reload**: Runtime system updates
- **Visual editor**: GUI for entity configuration
- **Profiler integration**: Built-in profiling
- **Network support**: Built-in networking patterns

### Ecosystem

- **More adapters**: Godot, MonoGame, etc.
- **Community samples**: User-contributed examples
- **Third-party tools**: Editor extensions
- **Tutorial content**: Video tutorials
