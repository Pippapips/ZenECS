ZenECS Core는 **계층화된 모듈 구조**를 가지고 있습니다:

```
Runtime/
├── Core/              # 핵심 인프라 (DI, 프리미티브)
├── Execution/         # 실행 엔진 (Kernel, Systems, Scheduling)
├── World/             # 월드 구현 (Entity, Component, Query)
├── Binding/           # 바인딩 시스템
├── Messaging/         # 메시지 버스
```
